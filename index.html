Welcome To bearBones
a free modern minimal fully responsive css framework for front end developers using a flat UI

bones was created by neccessity for those who want a headstart on development but are tired of sifting through over bloated frameworks.

this library is seperated into 3 files.

  1. bones-columns.css - contains code for page layouts
  2. bones-assets.css - contains code for assets i.e. buttons, text, inputs etc.
  3. custom.css - contains code for overriding sitewide colors and your custom css

use what you need remove what you dont, all styles can be customized by modifying the custom.css file.
